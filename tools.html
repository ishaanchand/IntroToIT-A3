Tools

The group has worked to produce mock-up versions of a website and mobile applications that would help illustrate how the recruitment project could be implemented. The group has used a variety of tools throughout the project;

  • Microsoft Teams for communication
  • GitHub for hosting the mock-up website (as well as the team page)
  • Atom and Visual Studio Coding for writing HTML (and other code)
  • Marvel (Prototypes) for the mobile application artefacts

(Source: Medium)

The GitHub commit trail does not accurately reflect the groups work as not everyone contributed via GitHub. Sam's work on the project webpage was mostly done in Visual Studio Coding and Nima's mobile app artefacts were created in Marvel.
